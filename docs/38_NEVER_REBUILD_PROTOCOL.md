# Never Rebuild Protocol

Whenever an AI receives a feature request,

perform these steps.

STEP 1

Locate route.

STEP 2

Locate page.

STEP 3

Locate parent layout.

STEP 4

Locate imported components.

STEP 5

Locate related hooks.

STEP 6

Locate stores.

STEP 7

Locate API.

STEP 8

Locate types.

STEP 9

Locate utilities.

Only then

decide whether implementation already exists.

---

Never regenerate

layout

forms

tables

dialogs

cards

buttons

without proving they cannot be reused.

---

Creating duplicate implementations is considered a bug.

---

Visual changes require explicit user approval.

---

Architecture changes require explicit user approval.

---

Folder restructuring requires explicit user approval.

---

Library replacement requires explicit user approval.
